Python

#python学习
首先python很有特点，相较于C语言或其他类编程来说语法极其简洁，应该比较容易学习，既支持面向过程编程，也支持面向对象编程，另外可以将其嵌入到C/C++程序中，从而提供脚本功能
一.os模块
当前Python脚本工作的目录路径: os.getcwd()
返回指定目录下的所有文件和目录名:os.listdir()
函数用来删除一个文件:os.remove()
删除多个目录：os.removedirs()
检验给出的路径是否是一个文件：os.path.isfile()
检验给出的路径是否是一个目录：os.path.isdir()
判断是否是绝对路径：os.path.isabs()
检验给出的路径是否真地存:os.path.exists()
返回一个路径的目录名和文件名:os.path.split() os.path.splitext()
获取路径名：os.path.dirname()
获取文件名：os.path.basename()
运行shell命令: os.system()
读取和设置环境变量:os.getenv() os.putenv()
给出当前平台使用的行终止符:os.linesep
创建多级目录：os.makedirs（r“c：\python\test”）
创建单个目录：os.mkdir（“test”）
获取文件属性：os.stat（file）
修改文件权限与时间戳：os.chmod（file）
终止当前进程：os.exit（）
获取文件大小：os.path.getsize（filename)

二.小乌龟画图

三.python中的数字类型
python中数字有四种类型：整数、布尔型、浮点数和复数。
int (整数) 如 1, 只有一种整数类型 int，表示为长整型
bool (布尔) 如 True
float (浮点数) 如 1.23、3E-3
complex (复数) 如 1 + 2j,2.3-4.6j
(给有经验的程序员的注释： 在 Python 中不用区分‘long int’类型。默认的整数类型可以任意长。)

四.引号
单引号
你可以用单引号指定字符串，如 ’Quote me on this’ 。所有的空白，即空格和制表 符都照原样保留。
双引号
在双引号中的字符串与单引号中的字符串的使用完全相同，例如 “What’s your name?” 。
三引号
利用三引号（”””or”’），你可以指示一个多行的字符串。你可以在三引号中自由 的使用单引号和双引号。例如：
‘’’This is a multi-line string. This is the first line.
This is the second line.
“What’s your name?,” I asked.
He said “Bond, James Bond.”
‘’’

五.标识符的命名
变量是标识符的例子。标识符是用来标识某样东西的名字。
在命名标识符的时候，你要遵循这些规则：
• 标识符的第一个字符必须是字母表中的字母（大写或小写）或者一个下划线 （‘’）。
• 标识符名称的其他部分可以由字母（大写或小写）、下划线（‘’）或数字（0-9） 组成。
• 标识符名称是对大小写敏感的。例如，myname 和 myName 不是一个标识符。注 意前者中的小写 n 和后者中的大写 N 。
• 有效标识符名称的例子有 i 、 __my_name 、 name_23 和 a1b2_c3 。
• 无效标识符名称的例子有 2things 、 this is spaced out 和 my-name 。

关于C语言的学习已经有一段时间，这段时间发现程序运行的功能只有你想不到，没有它做不到的。其中代码是基础，无论复杂程度，每一行代码都有它不可或缺的作用，而我们要做的就是把每一行，每一个思路都弄得清楚明白，这样才可以灵活运用，同时还要注意养成良好的习惯。而在这段时间的学习内，自己可以动手用程序解决平常出现的一些问题，当然也有一部分题目涉及范围太广，我还没有掌握，仍需继续努力。

一个有趣的编程 猜数字游戏：

#include<stdio.h>

#include<stdlib.h>
int main()
{
int d=1,e=0;
int a,b,t;
printf(“请输入1-10以内的整数,\n”);
while(d==1)
{
printf(“玩家的选择:”);
scanf(“%d”,&a);
do
{
if((a>=0)&&(a<11))
break;
else
{
printf(“error!\n”);
scanf(“%d”,&a);
}
}
while(e==0);
b=rand()%10+1;
printf(“npc:%d”,b);
t=a-b;
if(t==0)
printf(“Right!\n”);
else if(t<0)
printf(“Wrong!太小了\n”);
else if(t>0)
printf(“Wrong!太大了\n”);
}
return 0;
}

其中stdlib.h是标准库函数头文件名。
rand()函数生产随机数，查看资料发现srand()也是产生为随机数。

rand和srand用法及区别：

rand函数
函数头文件：stdlib.h
函数定义：int rand(void)
功能：产生一个随机数
而在严格意义上来讲，并不是真正的随机数，它采用了线性同余法算出随机数，即使用了一个公式。

srand函数
函数头文件：stdlib.h
含义：随机数发生器的初始化函数
定义：void srand（unsigned int seed）
功能：设置随机数种子
通常可以利用时间来改变系统种子值，可以实现真正意义上的随机。
