C语言学习

…
关于C语言的学习已经有一段时间，这段时间发现程序运行的功能只有你想不到，没有它做不到的。其中代码是基础，无论复杂程度，每一行代码都有它不可或缺的作用，而我们要做的就是把每一行，每一个思路都弄得清楚明白，这样才可以灵活运用，同时还要注意养成良好的习惯。而在这段时间的学习内，自己可以动手用程序解决平常出现的一些问题，当然也有一部分题目涉及范围太广，我还没有掌握，仍需继续努力。

简单的案例： 猜数字游戏：

#include<stdio.h>

#include<stdlib.h>
int main()
{
int d=1,e=0;
int a,b,t;
printf(“请输入1-10以内的整数,\n”);
while(d==1)
{
printf(“玩家的选择:”);
scanf(“%d”,&a);
do
{
if((a>=0)&&(a<11))
break;
else
{
printf(“error!\n”);
scanf(“%d”,&a);
}
}
while(e==0);
b=rand()%10+1;
printf(“npc:%d”,b);
t=a-b;
if(t==0)
printf(“Right!\n”);
else if(t<0)
printf(“Wrong!太小了\n”);
else if(t>0)
printf(“Wrong!太大了\n”);
}
return 0;
}
